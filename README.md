# PyRead #

Read source code from file

usage
======
```bash:
usage: read.py [-h] [-m METHOD] [-s STYLE] [-l] FILE

positional arguments:
  FILE                  Python File

options:
  -h, --help            show this help message and exit
  -m METHOD, --method METHOD
                        Print method with color
  -s STYLE, --style STYLE
                        Style coloring, default = "fruity"
  -l, --list-style      List valid of Style coloring
```
